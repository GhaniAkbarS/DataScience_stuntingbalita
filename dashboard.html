📊 Chapter 1: Kondisi Stunting di Indonesia

Indonesia menghadapi tantangan serius dalam masalah stunting. Berdasarkan data yang dianalisis, 24.4% balita mengalami stunting, yang berarti 1 dari 4 anak mengalami gangguan pertumbuhan.

👫 Chapter 2: Perbedaan Gender dalam Stunting

Analisis menunjukkan bahwa tidak ada perbedaan signifikan antara anak laki-laki dan perempuan dalam prevalensi stunting. Hal ini mengindikasikan bahwa faktor gender bukan penentu utama risiko stunting.

🎯 Chapter 3: Clustering Insight

Machine learning clustering mengidentifikasi 3 kelompok balita berdasarkan karakteristik fisik. Kelompok dengan risiko tinggi stunting dapat diidentifikasi untuk intervensi dini.

💡 Insights & Rekomendasi

Key Finding 1

Prevalensi stunting sebesar 24.4% masih di atas target WHO sebesar 20%, memerlukan intervensi yang lebih intensif.

Key Finding 2

Model machine learning mencapai akurasi 87.5% dalam prediksi status stunting, dapat digunakan untuk screening awal.

Key Finding 3

Clustering analysis mengidentifikasi 3 kelompok risiko yang dapat digunakan untuk personalisasi intervensi.

🎯 Rekomendasi Berbasis Data

Untuk Tenaga Kesehatan
  • Implementasi screening digital menggunakan model AI untuk deteksi dini stunting
  • Fokus intervensi pada kelompok usia 0-24 bulan yang menunjukkan risiko tertinggi
  • Monitoring rutin menggunakan Z-score WHO sebagai standar evaluasi
Untuk Peneliti
  • Eksplorasi faktor sosio-ekonomi sebagai variabel tambahan dalam model
  • Pengembangan model ensemble untuk meningkatkan akurasi prediksi
  • Validasi model pada dataset dari region geografis yang berbeda
Untuk Pembuat Kebijakan
  • Alokasi sumber daya berdasarkan pemetaan risiko dari clustering analysis
  • Pengembangan program intervensi terintegrasi untuk mencapai target WHO
  • Implementasi sistem monitoring digital berbasis data untuk evaluasi program
